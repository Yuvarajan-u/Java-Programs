Class: Noun 
eg: student, person, computer, HashMap

if it has more than one word we should do camel casing rule=> MyJavaProgram

First letter should be lowercase

if it has more than one word we should do like 
=> buttonClicked using uppercase for second word first letter

eg: actionPerformed,run,Print, Calculate,Compute

eg: student name => String sname="yuva";

eg: Runable, Readable, Remote

Everything Uppercase
